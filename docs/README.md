# Tidy Resize Images — documentation

Operator-focused docs for installing, running, and extending the
plugin.

## Getting started

- [**Brand new site**](use-cases/new-site.md) — install on a clean
  WordPress, set sensible defaults, and let new uploads be handled at
  arrival.
- [**Inherited site full of old, oversized images**](use-cases/inherited-site.md)
  — the bulk-update workflow. How to scan, protect logos and brand
  artwork, run the bulk page over hours or days, and recover from any
  mistakes.

## Reference

- [**WP-CLI commands**](wp-cli.md) — every command with examples.
  Mirrors the admin UI; useful for SSH-only operators and scripted
  pipelines.
- [**Action hooks and filters**](hooks-and-filters.md) — how to
  override the format decision tree from theme or plugin code.

## Where else to look

- The plugin's `readme.txt` covers WordPress.org-style metadata,
  description, and install steps.
- The `CHANGELOG.md` is the per-release record of what changed.
- For developers contributing to the plugin itself, `CLAUDE.md` and
  `.github/copilot-instructions.md` set out the coding standards;
  `dev-notes/` has the milestone tracker and architecture notes.
